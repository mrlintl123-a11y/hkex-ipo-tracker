# 港股 IPO 字段定义

## 基础字段（招股章程必含）

| 字段 | 英文 | 示例 | 说明 |
|---|---|---|---|
| 公司名称 | Issuer Name | 仙工智能 | 中英文 |
| 股票代码 | Stock Code | 06106.HK | 5 位数字 + .HK |
| 上市日期 | Listing Date | 2026-06-24 | 预期挂牌日 |
| 招股期 | Subscription Period | 6/15-6/18 | 含具体日期 |
| 截止日 | Closing Date | 2026-06-18 | 招股最后一天 |
| 发售价区间 | Offer Price Range | 18.2-21.0 港元 | 单一价 / 区间价 |
| 每手股数 | Board Lot Size | 200 股 | 港交所规定 |
| 入场费 | Entry Fee | 4,646 港元 | 每手 × 最高价 + 1% 费用 |
| 全球发售股数 | Total Offer Shares | 5,805.44 万股 | 含初始 + 调整 + 绿鞋 |
| 公开初始比例 | Initial Public Tranche | 5% / 10% | 通常 10% |
| 国配初始比例 | Initial Placing Tranche | 90% / 95% | 通常 90% |
| 基石投资者 | Cornerstone Investors | 高瓴/广发等 | 名单 + 金额 + 占比 |
| 是否有绿鞋 | Over-allotment Option | Yes 15% | 标准 15% |
| 保荐人 | Sponsors | 中金/摩根士丹利 | 联席保荐多家 |
| 是否 A+H | A+H Listing | Yes | A 股代码 + 名称 |

## 派生字段（计算或实时抓取）

| 字段 | 公式/来源 | 示例 |
|---|---|---|
| **公开手数** | `(全球发售股数 × 公开初始比例) ÷ 每手股数` | 5,805.44万 × 10% ÷ 200 = 2.9 万手 |
| **公开募资额** | `全球发售股数 × 公开初始比例 × 最高发售价` | 用于孖展倍数计算的分母 |
| **孖展认购倍数** | `券商借出孖展总额 ÷ 公开募资额` | 含 50-200 倍杠杆 |
| **公开认购倍数** | `（孖展+现金）÷ 公开募资额` | 配发结果公告中 |
| **国配认购倍数** | `国配申购量 ÷ 国配初始额度` | 配发结果公告中 |
| **回拨后公开比例** | 按回拨规则（见 ipo-mechanics.md） | 100 倍+→50% |
| **一手中签率** | 配发结果公告（"Allotment Results"）| 例：2% |
| **预期上市市值** | `发行后总股本 × 最高发售价` | 60.2 亿港元 |
| **数据可信度** | 见下表 | 🟢🟡🟠⚫ 四档 |

## 认购倍数数据可信度

| 标识 | 含义 | 适用场景 |
|---|---|---|
| 🟢 接近最终 | 已截止，配发结果未出 | 截止日当天收盘后 |
| 🟡 今日仍变 | 截止前 1 天 | 招股最后一天 |
| 🟠 仍有变数 | 截止前 2 天及以上 | 招股中期 |
| ⚫ N/A | 刚启动，暂无孖展 | 招股期首日 |

## 招股期阶段时间线

```
T-7: 招股期开始
T-2: 孖展数据开始累积
T-1: 孖展数据"决战日"（占总量 30-50%）
T  : 截止日 17:00 截止认购
T+1: 配发结果公告（18:00 后）
T+2: 资金解冻，可打新下一只
T+5~T+7: 挂牌上市
```
